### BINATEXT ###

`` A simple program written in python for binary texting. ``

# Usage: 

===============================
1. Run the exe file/Python script
2. Enter your message of desire
3. If you want, you can press alt on your keyboard, wait 3 seconds on a tab in which you want the binary
to be typed in, and the program will do it for you. You can also just get the results without the typing
process, by pressing Esc, as instructed
===============================
